Processing Mirror library

Purpose

A Processing utility to draw the mirror image (mirage) of a PGraphics object onto that object. A mirror can be constructed with an angle, position, tint, etc. and then later used to draw the mirage of an object. The position is always set relative to the center of the image.

The mirage can be drawn as a PImage, or pixel by pixel in a x,y loop, or by iterating the pixels[] array of the source.



To see what you can do with this, watch some of the animations at https://www.instagram.com/studio.pike
